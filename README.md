# MC750-UI-que-energIA

Abra o terminal e navegue até a pasta do seu projeto.
Crie a venv:
```bash
python3 -m venv .venv
```
Ative a venv:
```bash
source .venv/bin/activate
```
Instale o pacote openai:
```bash
pip install openai
```
Instale o pacote dotenv, para importação das KEYs confidenciais:
```bash
pip install dotenv
```
